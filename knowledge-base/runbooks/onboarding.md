# Engineer Onboarding

## Day 1
- [ ] Get access to GitHub org
- [ ] Get access to Linear workspace
- [ ] Get access to Slack channels
- [ ] Set up local development environment
- [ ] Read system-overview.md and tech-stack.md

## Week 1
- [ ] Complete first good-first-issue PR
- [ ] Pair program with a team member
- [ ] Review the architecture decision records (ADRs)
- [ ] Understand the deploy process (deploy-production.md)

## Week 2
- [ ] Take a shadow on-call shift
- [ ] Review the incident response runbook
- [ ] Set up monitoring dashboards
